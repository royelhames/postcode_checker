== README

This README documents steps necessary to get the
application up and running.

This app running on rails version 4.2.11.1
There is no database , to deploy:
- download the app 
- bundle install
- Fire up your rails server on port 3000 
- browse to http://localhost:3000

Enter a postcode and click return to check if such postcode is servable by SH24 

To run unit tests :
/usr/local/bin/ruby /usr/local/bin/rake test test/models/postcode_test.rb



* ...

<tt>rake doc:app</tt>.
